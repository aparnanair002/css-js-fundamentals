HTTP and Forms

What Is HTTP?

The Hypertext Transfer Protocol (HTTP) is the protocol used for communication between web clients (browsers) and servers. It is stateless, meaning each request must contain all information needed to understand it. The server responds with a status and content.

How HTTP Works

  • A client opens a TCP connection to the server‚Äôs HTTP port (typically 80 for HTTP, 443 for HTTPS).
  • The client sends a request line like GET /path HTTP/1.1.
  • The server replies with a status line (e.g., HTTP/1.1 200 OK), headers, and a body.

HTTP Methods

  • GET ‚Äî Retrieve a resource.
  • POST ‚Äî Send data to the server.
  • PUT ‚Äî Replace or create a resource.
  • DELETE ‚Äî Remove a resource.

Browsers and HTTP

  • Browsers request HTML pages and then automatically request referenced resources (CSS, JS, images).
  • Headers provide metadata (e.g., Content‚ÄëType, Content‚ÄëLength).
  • Status codes indicate success (200s), redirection (300s), client errors (400s), and server errors (500s).

Forms

HTML forms allow users to submit data. Form controls (<input>, <textarea>, <select>) send data via query strings or request bodies.

The Fetch API

JavaScript can make HTTP requests using the fetch() API, which returns a promise resolving to a response object. 


fetch("example/data.txt")
  .then(response => response.text())
  .then(text => console.log(text));

Form Fields

Form fields have properties (e.g., value) and fire events (input, change). You can prevent default submission with event handlers.

Security

  • Browsers restrict cross‚Äëdomain HTTP requests for security.
  • CORS headers (e.g., Access‚ÄëControl‚ÄëAllow‚ÄëOrigin) can allow cross‚Äëdomain access.
  • HTTPS encrypts HTTP to protect data.

Appreciating HTTP

Understanding HTTP helps in web development, debugging, and building web applications.

Focus

  • The focus() method sets focus to a form field.
  • The blur() method removes focus from a form field.

Text Fields

Text fields allow users to input text data. They can be single-line (<input type="text">) or multi-line (<textarea>).

Check boxes and radio buttons

Check boxes and radio buttons allow users to select options.

select

The <select> element creates a drop-down list of options.

File Uploads

The <input type="file"> element allows users to upload files.
