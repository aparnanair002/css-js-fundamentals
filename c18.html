HTTP and Forms

What Is HTTP?

The Hypertext Transfer Protocol (HTTP) is the protocol used for communication between web clients (browsers) and servers. It is stateless, meaning each request must contain all information needed to understand it. The server responds with a status and content. :contentReference[oaicite:1]{index=1}

How HTTP Works

  • A client opens a TCP connection to the server’s HTTP port (typically 80 for HTTP, 443 for HTTPS). :contentReference[oaicite:2]{index=2}
  • The client sends a request line like GET /path HTTP/1.1. :contentReference[oaicite:3]{index=3}
  • The server replies with a status line (e.g., HTTP/1.1 200 OK), headers, and a body. :contentReference[oaicite:4]{index=4}

HTTP Methods

  • GET — Retrieve a resource. :contentReference[oaicite:5]{index=5}
  • POST — Send data to the server. :contentReference[oaicite:6]{index=6}
  • PUT — Replace or create a resource. :contentReference[oaicite:7]{index=7}
  • DELETE — Remove a resource. :contentReference[oaicite:8]{index=8}

Browsers and HTTP

  • Browsers request HTML pages and then automatically request referenced resources (CSS, JS, images). :contentReference[oaicite:9]{index=9}
  • Headers provide metadata (e.g., Content‑Type, Content‑Length). :contentReference[oaicite:10]{index=10}

Forms

HTML forms allow users to submit data. Form controls (<input>, <textarea>, <select>) send data via query strings or request bodies. :contentReference[oaicite:11]{index=11}

The Fetch API

JavaScript can make HTTP requests using the fetch() API, which returns a promise resolving to a response object. :contentReference[oaicite:12]{index=12}


fetch("example/data.txt")
  .then(response => response.text())
  .then(text => console.log(text));

Security

  • Browsers restrict cross‑domain HTTP requests for security. :contentReference[oaicite:13]{index=13}
  • CORS headers (e.g., Access‑Control‑Allow‑Origin) can allow cross‑domain access. :contentReference[oaicite:14]{index=14}
  • HTTPS encrypts HTTP to protect data. :contentReference[oaicite:15]{index=15}

Working With Form Fields

  • Form fields have properties like value and fire events (e.g., input, change). :contentReference[oaicite:16]{index=16}
  • You can prevent default submission with event handlers. :contentReference[oaicite:17]{index=17}
