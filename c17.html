Drawing on Canvas


<svg xmlns="http://www.w3.org/2000/svg"> <circle r="50" cx="50" cy="50" fill="red"/> <rect x="120" y="5" width="90" height="90" stroke="blue" fill="none"/> </svg>

Introduction

The <canvas> element provides a bitmap area that can be drawn on using JavaScript. Canvas graphics are pixel-based and do not retain object structure like SVG.

To draw, you must first get the 2D rendering context:


let canvas = document.querySelector("canvas");
let ctx = canvas.getContext("2d");

Drawing Rectangles

  • fillRect(x, y, w, h) — draws filled rectangle
  • strokeRect(x, y, w, h) — draws rectangle outline
  • Set color using fillStyle or strokeStyle

ctx.fillStyle = "red";
ctx.fillRect(10, 10, 100, 50);

ctx.strokeStyle = "blue";
ctx.lineWidth = 5;
ctx.strokeRect(135, 5, 50, 50);

Paths and Lines

  • Use beginPath() to start a new path
  • moveTo(x, y) sets starting point
  • lineTo(x, y) adds a line to the path
  • stroke() draws the path

ctx.beginPath();
ctx.moveTo(10, 10);
ctx.lineTo(100, 100);
ctx.stroke();

Curves

  • Quadratic curve: quadraticCurveTo(cpx, cpy, x, y)
  • Cubic Bezier curve: bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y)

// Quadratic curve
ctx.beginPath();
ctx.moveTo(10, 90);
ctx.quadraticCurveTo(60, 10, 90, 90);
ctx.stroke();

// Bezier curve
ctx.beginPath();
ctx.moveTo(10, 10);
ctx.bezierCurveTo(10, 10, 90, 10, 50, 90);
ctx.stroke();

Text

  • fillText(text, x, y) — fills text
  • strokeText(text, x, y) — outlines text
  • Set font using ctx.font

ctx.font = "28px Georgia";
ctx.fillText("Hello Canvas", 10, 50);

Images

  • Draw images with drawImage(img, x, y)
  • Can crop or resize using extra parameters

let img = document.createElement("img");
img.src = "img.png";
img.addEventListener("load", () => {
  ctx.drawImage(img, 10, 10);
});

Transformations

  • translate(x, y) — moves coordinate system
  • scale(x, y) — stretches/shrinks drawing
  • rotate(angle) — rotates drawing
  • Use save() and restore() to manage state

ctx.save();
ctx.translate(50,50);
ctx.rotate(Math.PI / 4);
ctx.restore();

Animation

  • Use clearRect() to clear canvas each frame
  • Use requestAnimationFrame() for smooth animation
  • Redraw all elements on each frame

function animate() {
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  // draw shapes here
  requestAnimationFrame(animate);
}
animate();

Canvas vs SVG

  • Canvas is pixel-based; fast for dynamic graphics
  • SVG is vector-based; retains shape objects for manipulation
  • Use Canvas for games and free-form drawing, SVG for diagrams and scalable graphics

Summary

  • Canvas is a powerful 2D drawing tool in the browser
  • Draw shapes, lines, curves, text, and images
  • Transformations and animations are done via JavaScript
  • Event loop and requestAnimationFrame handle interactive graphics efficiently
  • Choose Canvas or SVG based on performance and interactivity needs

Exercises

  • Pie Chart: Draw a pie chart with colored slices using arc() and loops.
  • Bouncing Ball: Animate a ball moving across the canvas with collision detection.
  • Drawing App: Implement free-hand drawing with mouse events.
