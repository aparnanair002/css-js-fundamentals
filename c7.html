Program Structure — Summary

“If we want a machine to be intelligent, it needs to be able to adjust its behavior…”
— Marvin Minsky

This chapter introduces how programs are structured in JavaScript. Programming is about using simple building blocks (values, variables, operators) and organizing them into meaningful structures.

Expressions & Statements

An expression produces a value. A statement performs an action.

Example of an expression in a statement:

1;
!false;

A real program is usually a series of statements.

Bindings (Variables)

A binding stores a value and lets you refer to it later:

let caught = 5 * 5;

Bindings can be updated to point to new values:

let mood = "light";
mood = "dark";

They do not “contain” values — they simply point to them.

Keywords: let, const, var

  • let — changeable binding
  • const — binding that cannot be reassigned
  • var — older keyword, behaves differently in scopes

Binding Names

Must start with a letter, $, or _. Examples:

archenemy, $result, _hidden

Not allowed:

2fast, my-variable

Some words are reserved (example: let, class, return).

Environment

The environment is the collection of bindings that exist at a given time. Some are built-in:

console.log("Hello");

You access them without creating them.

Functions

A function is a program wrapped in a value. You call it using parentheses:

alert("Hello!");
console.log(Math.max(3, 11));

Return Values

Some functions return values, others (like alert()) don't.

Control Flow

By default, statements run top to bottom. Control flow statements change that.

Conditional Execution

if runs code only when something is true.

let num = 10;
if (num > 5) {
  console.log("Large");
} else {
  console.log("Small");
}

Looping

Loops repeat actions.

while loop:

let number = 0;
while (number <= 12) {
  console.log(number);
  number = number + 2;
}

do…while loop:

do {
  var name = prompt("Who?");
} while (!name);

for loop:

for (let i = 0; i <= 12; i = i + 2) {
  console.log(i);
}

Breaking out

for (let current = 20; ; current++) {
  if (current % 7 == 0) {
    console.log(current);
    break;
  }
}

Updating Bindings

Use operators like += or ++:

let counter = 0;
counter += 2;
counter++;

Dispatching on a Value: switch

switch (weather) {
  case "rainy":
    console.log("Bring umbrella");
    break;
  case "sunny":
    console.log("Wear sunglasses");
    break;
  default:
    console.log("Unknown weather");
}

Capitalization

JavaScript is case-sensitive. Convention uses camelCase:

thisIsCamelCase
addedTaxRate

   comment */

Summary

This chapter introduced the basic structures of JavaScript programs—bindings, functions, control flow, loops, and conditionals—showing how simple tools combine into powerful logic.
