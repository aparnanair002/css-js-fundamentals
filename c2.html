üí° JavaScript Summary

Program Structure

A JavaScript program is a sequence of statements that the computer executes one by one. Statements are built using expressions, variables, keywords, and operators.

üìú Statements & Semicolons

  • Each statement performs an action, such as a calculation or a variable declaration.
  • Statements are separated by semicolons (;).
  • Line breaks often act as semicolons automatically (Automatic Semicolon Insertion), but relying on it is risky.

üî£ Expressions

  • An expression is a piece of code that produces a value.
  • Examples: 5 * 10, "Hello" + "World", x > 10
  • Expressions can be combined to form more complex statements.

üì¶ Variables

  • Used to store values using let, const, or var.
  • let ‚Äì block-scoped, reassignable.
  • const ‚Äì block-scoped, cannot be reassigned.
  • var ‚Äì function-scoped (older way, avoid using).
let age = 25;
const name = "Aparna";
var city = "Kochi";

üß≠ Control Flow

By default, code executes from top to bottom. Control flow statements change this order.

  • Conditional statements ‚Äì decide based on conditions.
  • Loops ‚Äì repeat actions.
  • Functions ‚Äì group code to be reused.

üîÄ Conditional Execution

if (condition) {
  // code runs if true
} else if (anotherCondition) {
  // runs if first is false
} else {
  // runs otherwise
}
  • if ‚Äì executes code if the condition is true.
  • else ‚Äì handles false conditions.
  • else if ‚Äì checks multiple conditions.
  • switch ‚Äì alternative for multiple if checks.

üîÅ Loops

  • while ‚Äì repeats while the condition is true.
  • do...while ‚Äì runs at least once, then repeats.
  • for ‚Äì common loop with initialization, condition, and update.
for (let i = 0; i < 5; i++) {
  console.log(i);
}

üß© Functions

  • Encapsulate reusable code.
  • Declared using function keyword or arrow syntax (=>).
function greet(name) {
  console.log("Hello, " + name);
}

const add = (a, b) => a + b;

üö™ Returning Values

  • return sends a value back from a function.
  • Stops further execution inside the function.
function square(x) {
  return x * x;
}

üß† In Essence

Program structure defines how your JavaScript code is organized and executed. It includes statements, expressions, variables, conditions, loops, and functions ‚Äî all working together to give logic and order to your code.
