Color Manipulation with CSS Filters
CSS filters allow you to apply visual effects to elements, such as images and backgrounds, directly through CSS. They can be used to adjust colors, blur images, and create various artistic effects without needing to edit the original image files. hue rotate(angle): Rotates the hue of the colors in the element by the specified angle (in degrees). This can create interesting color shifts. //backdrop filters .box { border-radius: 10% 20% 30% 40% / 50% 60% 70% 80%; } The first 4 numbers are all horizontal radiuses. We specify them in clockwise order, starting from the top-left corner. The last 4 numbers are the vertical radiuses, also specified in clockwise order from the top-left. box-shadow comes with several parameters: offset-x | offset-y | blur-radius | spread-radius | color

Gradients
Gradients are created using the background-image property in CSS. There are two main types of gradients: linear gradients and radial gradients. .box { background-image: linear-gradient(to right, color1, color2); /* Is equivalent to: */ background-image: linear-gradient(90deg, color1, color2); } background-image: radial-gradient( 50% 100%, deeppink, gold ); background-image: conic-gradient( deeppink, coral, gold, coral, deeppink /* <-- same color! */ ); clip-path:polygon(50% 0%, 0% 100%, 100% 100%);
Smooth Scrolling
To enable smooth scrolling for the entire webpage, you can use the following CSS: html { scroll-behavior: smooth; } also, @media (prefers-reduced-motion: no-preference) { html { scroll-behavior: smooth; } }
